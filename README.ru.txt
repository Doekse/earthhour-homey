Час Земли, организованный WWF, — это глобальное общественное движение, объединяющее людей для действий по экологическим проблемам и защиты планеты. Каждый год в последнюю субботу марта миллионы людей объединяются в 20:30 по местному времени, чтобы выключить свет и посвятить час Земле. Этот символический жест вдохновляет на коллективные действия по защите нашей планеты.

Это приложение Homey позволяет легко присоединиться к Самому Важному Часу для Земли. Автоматически определяйте начало и конец Часа Земли в вашем часовом поясе и используйте триггеры Flow для выключения света и устройств. Создавайте условия для проверки, является ли текущее время Часом Земли, чтобы ваш дом мог участвовать в этом глобальном движении. Посетите earthhour.org для получения дополнительной информации.
