La Hora del Planeta, organizada por WWF, es un movimiento ciudadano global que une a las personas para actuar sobre los problemas ambientales y proteger el planeta. Cada año, el último sábado de marzo, millones de personas se unen a las 20:30 hora local para apagar sus luces y dar una hora por la Tierra. Este gesto simbólico inspira acción colectiva para proteger nuestro planeta.

Esta aplicación Homey hace que sea muy fácil unirse a la Mayor Hora por la Tierra. Detecta automáticamente cuándo comienza y termina la Hora del Planeta en tu zona horaria, y usa los disparadores Flow para apagar tus luces y dispositivos. Crea condiciones para verificar si es actualmente la Hora del Planeta, para que tu hogar pueda participar en este movimiento global. Visita earthhour.org para más información.
